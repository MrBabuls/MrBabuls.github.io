Lost In The Morgue

  • November 6, 2025

Project description

I continued working on a game jam project that was originally created as part of Xamk Game Jam 2024, which took place from April 2024. We decided that we want to continue developing the project further. The game’s genre will be horror and the setting will be in Kotka.

Description of the task

My task was to create 3D models for the game. This included modelling the characters, monsters, and the environment assets.

Description of methods

I first created a reference image of the main character and showed it to the rest of the team. After they approved it, I started modelling the character.

3D model

Above, you can see the reference image and the 3D character that I modeled.

Problems and solutions

The main problem is balancing my time so that I could continue working on my personal projects. I have not found a solution for this yet. I also haven’t managed to finish the 3D model. The character is still missing the head, the hands, and of course the clothing.

Final assessment

I am not satisfied with the fact that I haven’t had time to continue this project. I have prioritized work and school tasks first and when I finally have free time, I want to spend it away from the computer. Because of that, my personal projects have been left untouched.
