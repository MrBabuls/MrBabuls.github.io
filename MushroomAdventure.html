Project description

Mushroom Adventure is a 3D adventure game currently in development by myself and Tarita. As of now, we've laid the groundwork with a detailed Game Design Document (GDD) and have commenced work on player movement and dialogue mechanics. Additionally, I've begun modeling both the player character and their sword.

Description of the task

As part of my role, I am responsible for the programming and modeling of the game's characters.

Problems and solutions

The biggest problem has been finding time for developing the project. The plan is to continue working on the project throughout the summer.
